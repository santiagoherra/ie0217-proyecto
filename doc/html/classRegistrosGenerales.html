Descripción detallada

En esta clase se encuentran los métodos utilizados para todo lo relacionado con registros generales y personales.

Los métodos de esta clase acceden a la base de datos para actualizar y obtener de forma detallada, toda la información referente al historial de transacciones, préstamos, créditos de depósito a plazo, y cuentas de clientes que han sido registradas en la base de datos del banco. De esta forma, se ofrece un historial de acciones que puede ser útil para el personal del banco y también sus clientes.

Este método accede a la tabla registros de la base de datos para guardar las transacciones exitosas realizadas en el banco.

Parámetros
tipo_transaccionIndica que tipo de transacción se efectuó (retiro, depósito, etc)
fecha_transaccionFecha en la que se realizó la transacción
denominacionIndica el tipo de moneda que se utilizó para realizar la transacción
cliente_origen_cedulaNúmero de cédula del cliente que emitió la transferencia
cliente_destino_cedulaNúmero de cédula del cliente que recibió la transferencia (en algunos casos es el mismo que el cliente origen)
montoBaseMonto que estuvo involucrado en la transferencia
